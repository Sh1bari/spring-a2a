# Compatibility

This document is the source of truth for supported release combinations.

## Current Support Matrix

| Spring A2A | A2A Java SDK | Spring Boot | Java |
| --- | --- | --- | --- |
| 0.1.x | 1.1.x | 3.5.x | 17+ |

## Notes

- The project currently targets Java 17 as the minimum runtime.
- Spring Boot and A2A Java SDK versions are pinned in the root `pom.xml`.
- Release branches may move forward independently, but the support matrix should stay explicit here.
